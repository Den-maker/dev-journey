# Урок 00: Введение в Go

**Статус:** ✅ Завершён

## Что изучили

- Историю создания Go и его философию
- Ключевые особенности языка (горутины, компиляция, типизация)
- Где Go сильнее всего и где его не стоит использовать
- Сравнение с Python, JavaScript, Java
- Типичные проекты на Go

## Ключевые идеи

- Go создан для **простоты** и **скорости разработки**
- Один исполняемый файл = лёгкий деплой
- Горутины = встроенная многопоточность
- Идеален для веб-серверов, API, микросервисов, CLI утилит

## Философия Go

> "Простота важнее гибкости"  
> "Не пиши умный код, пиши простой код"  
> "Обрабатывай ошибки явно"
