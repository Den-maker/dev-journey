## Ключевые команды
```bash
go run main.go          # скомпилировать и запустить
go build main.go        # скомпилировать в исполняемый файл
go fmt main.go          # отформатировать код
go version              # проверить версию Go
```

## Важные моменты

- `package main` + `func main()` = точка входа в программу
- Go компилируется в один исполняемый файл без зависимостей
- VS Code с расширением Go автоматически форматирует код при сохранении
- Неиспользуемые переменные и импорты = ошибка компиляции
