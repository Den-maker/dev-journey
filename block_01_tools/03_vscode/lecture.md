# VS Code для разработки на Go

## Установка расширения Go

### Через интерфейс VS Code
1. Открыть панель расширений: `Ctrl+Shift+X`
2. Найти "Go" (автор: Go Team at Google)
3. Нажать "Install"

### Дополнительные инструменты
При первом открытии `.go` файла VS Code предложит установить дополнительные инструменты Go.

**Рекомендуется установить всё** — нажать "Install All"

Эти инструменты включают:
- `gopls` — языковой сервер (автодополнение, навигация)
- `dlv` — отладчик
- `staticcheck` — линтер для проверки кода
- и другие полезные утилиты

## Встроенный терминал

### Открыть/закрыть терминал
```
Ctrl+`  (backtick, клавиша с буквой Ё)
```

### Создать новый терминал
```
Ctrl+Shift+`
```

Терминал открывается в корне открытой папки — очень удобно для команд Go.

## Полезные горячие клавиши

### Общие
```
Ctrl+S          — Сохранить файл
Ctrl+Shift+S    — Сохранить все файлы
Ctrl+P          — Быстрый поиск файлов по имени
Ctrl+Shift+P    — Палитра команд (поиск любой команды VS Code)
Ctrl+B          — Показать/скрыть боковую панель
```

### Редактирование
```
Ctrl+/          — Закомментировать/раскомментировать строку
Ctrl+D          — Выделить следующее вхождение слова
Alt+↑/↓         — Переместить строку вверх/вниз
Shift+Alt+↓     — Дублировать строку вниз
Ctrl+Shift+K    — Удалить строку
```

### Навигация
```
Ctrl+G          — Перейти к строке по номеру
F12             — Перейти к определению
Alt+←/→         — Назад/вперёд по истории переходов
```

### Markdown
```
Ctrl+Shift+V    — Открыть предпросмотр Markdown
Ctrl+K V        — Открыть предпросмотр сбоку
```

## Настройки для Go

После установки расширения Go, VS Code автоматически:
- Форматирует код при сохранении
- Показывает ошибки в реальном времени
- Предоставляет автодополнение
- Позволяет запускать код через кнопку "Run" или сочетание клавиш

### Запуск Go кода в VS Code
```
Ctrl+F5         — Запустить без отладки
F5              — Запустить с отладчиком
```

## Полезные настройки (опционально)

Открыть настройки: `Ctrl+,`

Рекомендуемые настройки для Go:
```json
{
  "editor.formatOnSave": true,
  "go.formatTool": "goimports",
  "go.lintOnSave": "workspace"
}
```

Но обычно расширение Go настраивает всё правильно по умолчанию.

## Работа с терминалом

В VS Code удобно использовать встроенный терминал для команд Go:
- Не нужно переключаться между окнами
- Терминал всегда открыт в правильной директории
- Можно иметь несколько терминалов одновременно

**Наше соглашение:**
- Встроенный терминал VS Code → для команд Go (`go run`, `go build`)
- Git Bash (отдельное окно) → для команд Git

## Snippets (сниппеты)

VS Code поддерживает быстрые шаблоны кода. После установки расширения Go будут доступны:

Напечатать `pkgm` и нажать `Tab` → создаст `package main`
Напечатать `ff` и нажать `Tab` → создаст fmt.Printf()

И многие другие. Они появляются в автодополнении.
