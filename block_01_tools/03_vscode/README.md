# Урок 03: VS Code

**Статус:** ✅ Завершён

## Что изучим

- Установка расширения Go
- Установка дополнительных инструментов Go
- Работа со встроенным терминалом
- Основные горячие клавиши
- Запуск Go программ из VS Code

## Ключевые моменты

- Расширение: **Go** от Go Team at Google
- При первом открытии `.go` файла установить все предложенные инструменты
- Терминал VS Code (`Ctrl+``) — для команд Go
- Git Bash (отдельное окно) — для команд Git

## Горячие клавиши (самые важные)
```
Ctrl+`          — Терминал
Ctrl+S          — Сохранить
Ctrl+P          — Поиск файлов
Ctrl+Shift+V    — Markdown предпросмотр
```
