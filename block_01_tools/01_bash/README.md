# Урок 01: Bash Commands

**Статус:** ✅ Завершён

## Что изучили

- Навигация по файловой системе (pwd, cd)
- Просмотр содержимого (ls с различными флагами)
- Создание файлов и папок (mkdir, touch)
- Фигурные скобки для множественных операций
- Удаление файлов и папок (rm)
- Открытие VS Code из терминала (code)

## Ключевые моменты

- Git Bash использует Linux-синтаксис команд
- Диски обозначаются как `/d/`, `/c/` а не `D:`, `C:`
- В фигурных скобках `{item1,item2}` не должно быть пробелов
- Git не отслеживает пустые папки — используем `.gitkeep`
