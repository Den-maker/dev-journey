# Bash Commands - Основные команды

## Навигация

### Узнать текущее расположение
```bash
pwd
```

### Перейти в другую папку
```bash
cd /d/go                    # абсолютный путь
cd dev-journey              # относительный путь
cd ..                       # на уровень выше
cd ~                        # в домашнюю папку пользователя
```

### Перейти на другой диск
```bash
cd /d/                      # на диск D
cd /c/Users/                # на диск C
```

## Просмотр содержимого

### Посмотреть файлы и папки
```bash
ls                          # обычный список
ls -a                       # показать скрытые файлы (начинаются с точки)
ls -l                       # подробный список с правами и размерами
ls -la                      # и скрытые, и подробно
```

### Посмотреть содержимое конкретной папки
```bash
ls block_01_tools/
ls -a block_01_tools/01_bash/
```

## Создание файлов и папок

### Создать одну папку
```bash
mkdir folder_name
```

### Создать вложенные папки сразу
```bash
mkdir -p parent/child/grandchild
```

### Создать несколько папок одной командой
```bash
mkdir -p block_01_tools/{01_bash,02_github,03_vscode}
```
**Важно:** внутри фигурных скобок не должно быть пробелов после запятых!

### Создать файл
```bash
touch filename.txt
touch block_01_tools/01_bash/lecture.md
```

### Создать файл в нескольких папках сразу
```bash
touch {folder1,folder2,folder3}/.gitkeep
touch block_01_tools/{01_bash,02_github}/.gitkeep
```

## Удаление

### Удалить файл
```bash
rm filename.txt
```

### Удалить несколько файлов по маске
```bash
rm *.gitkeep                # все файлы с расширением .gitkeep
```

### Удалить папку
```bash
rm -r folder_name           # -r означает рекурсивно (с содержимым)
```

## Открытие в программах

### Открыть текущую папку в VS Code
```bash
code .
```

### Открыть конкретный файл в VS Code
```bash
code README.md
```

## Полезные команды

### Очистить экран терминала
```bash
clear
# или Ctrl+L
```

### Автодополнение
Начните печатать название файла или папки и нажмите **Tab** — bash автоматически дополнит название.

### История команд
Стрелка вверх/вниз — перемещение по истории ранее введённых команд.
